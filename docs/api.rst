.. py:currentmodule:: graphpatch

API
===
The public API to ``graphpatch`` consists of :class:`PatchableGraph`, which is a wrapper around :class:`Module`,
and :class:`Patch <graphpatch.patch.Patch>`, which is a compact representation of an intervention
to perform on an intermediate Tensor value.
